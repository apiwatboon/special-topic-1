# 🐙 รู้จักและใช้งาน GitHub — สื่อการสอน Special Topic 1

สื่อการสอนแบบเว็บไซต์ (interactive) สำหรับรายวิชา **Special Topic 1**
สาขาวิชาวิศวกรรมคอมพิวเตอร์ — หัวข้อ **การใช้งาน GitHub** (สัปดาห์ที่ 1)

> GitHub คืออะไร · มาจากไหน · ใช้อย่างไร · มีประโยชน์อย่างไร — ครบในเว็บเดียว

---

## 📖 เนื้อหาในสื่อ

1. GitHub คืออะไร
2. ประวัติความเป็นมา
3. Git ≠ GitHub (ความแตกต่าง)
4. ประโยชน์ของ GitHub
5. แนวคิดและคำศัพท์พื้นฐาน
6. การสมัครสมาชิก (ทีละขั้น)
7. การอัปโหลดโปรเจกต์ (แบบหน้าเว็บ + แบบคำสั่ง Git)
8. คำสั่ง Git พื้นฐาน
9. Workflow การทำงานในแต่ละวัน
10. ฟีเจอร์เด่นอื่น ๆ
11. การเผยแพร่เว็บด้วย GitHub Pages
12. แบบทดสอบความเข้าใจ (โต้ตอบได้)
13. ใบงาน / Lab
14. แหล่งเรียนรู้เพิ่มเติม

## 🗂️ โครงสร้างไฟล์

```
.
├── index.html   # โครงสร้างเนื้อหาเว็บ
├── style.css    # ดีไซน์และการจัดวาง
├── script.js    # การโต้ตอบ (เมนู, แท็บ, ปุ่มคัดลอก, แบบทดสอบ)
└── README.md    # ไฟล์นี้
```

## 🚀 วิธีเปิดใช้งาน

**เปิดในเครื่อง:** ดับเบิลคลิกไฟล์ `index.html` ได้เลย ไม่ต้องติดตั้งอะไร

**เผยแพร่ออนไลน์ด้วย GitHub Pages:**
1. อัปโหลดไฟล์ทั้งหมดขึ้น repository บน GitHub
2. ไปที่ **Settings → Pages**
3. เลือก branch `main` โฟลเดอร์ `/ (root)` แล้วกด **Save**
4. เว็บจะออนไลน์ที่ `https://USERNAME.github.io/REPO-NAME`

- เนื้อหาทั้งหมดเป็นภาษาไทย เหมาะกับการบรรยายในชั้นเรียน
- มีแบบทดสอบและใบงานท้ายบทให้นักศึกษาฝึกปฏิบัติ
- ปรับแก้เนื้อหา/สถิติ/ใบงานได้อิสระตามบริบทของรายวิชา
