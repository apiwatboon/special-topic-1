สาขาวิชาวิศวกรรมคอมพิวเตอร์

Special Topic 1 · สื่อการสอน

หัวข้อพิเศษทางวิศวกรรมคอมพิวเตอร์ — เทคโนโลยีปัจจุบันที่ยังไม่อยู่ในหลักสูตร แต่ต้องรู้ก่อนไปทำงานจริง เรียนรู้จากการมอง ดู อ่าน และลงมือทำตาม ครบ 16 สัปดาห์

01 เรียนแล้ว

รู้จักและใช้งาน GitHub

Git ≠ GitHub, การสมัคร, อัปโหลดโปรเจกต์, คำสั่งพื้นฐาน, ทำงานเป็นทีมด้วย Pull Request และทำเว็บด้วย GitHub Pages

เปิดบทเรียน
02 เรียนแล้ว

Vibe Coding — เขียนโค้ดยุค AI

เขียนโปรแกรมด้วยการ "บอกสิ่งที่อยากได้" ให้ AI ช่วยสร้าง ส่องสมอง AI แบบ 3 มิติ ลองเล่นซิมูเลเตอร์จริง และฝึกตั้งคำสั่ง (prompt) แบบมืออาชีพ

เปิดบทเรียน
03 ใหม่ล่าสุด

Vibe Coding ทั้งโปรเจกต์ — เว็บง่ายถึงซับซ้อน

ยกระดับจากสั่งทีละชิ้น สู่การสร้างเว็บทั้งโครงการ ผ่าน 2 เคสจริง: เว็บหน้าเดียวจบใน 4 คำสั่ง และเว็บแอปซับซ้อนที่เดินเป็น Milestone

เปิดบทเรียน
04 เร็ว ๆ นี้

Deploy เว็บสู่โลกจริง

จาก localhost สู่ URL ที่ใครก็เปิดได้: GitHub Pages ขั้นสูง, Netlify/Vercel, โดเมนของตัวเอง, HTTPS และเช็กลิสต์ก่อนปล่อยเว็บให้โลกเห็น

ยังไม่เปิด
05 เร็ว ๆ นี้

ต่อพลัง AI เข้าแอปของเรา — LLM API

เรียกใช้โมเดล AI ผ่าน API สร้างแชตบอตของตัวเองตัวแรก เข้าใจ token กับค่าใช้จ่าย และวิธีเก็บ API key ให้ปลอดภัย

ยังไม่เปิด
06 เร็ว ๆ นี้

AI Agents & MCP — ให้ AI ลงมือทำแทนเรา

จากแชตบอตที่ "ตอบ" สู่เอเจนต์ที่ "ทำ": AI ที่ใช้เครื่องมือได้เอง ทำงานหลายขั้นตอนอัตโนมัติ และมาตรฐาน MCP ที่กำลังเปลี่ยนวงการ

ยังไม่เปิด
07 เร็ว ๆ นี้

ฐานข้อมูล + Backend สำเร็จรูป

เก็บข้อมูลจริงให้เว็บแอปด้วย Supabase/Firebase: ตาราง, ระบบล็อกอิน, API อัตโนมัติ — โดยไม่ต้องเขียนเซิร์ฟเวอร์เองสักบรรทัด

ยังไม่เปิด
08 เร็ว ๆ นี้

Docker & Containers

แก้ปัญหาโลกแตก "บนเครื่องผมรันได้นะ": จับแอปพร้อมทุกอย่างที่มันต้องใช้ใส่กล่อง แล้วรันที่ไหนก็ได้ผลเหมือนกันเป๊ะ

ยังไม่เปิด
09 เร็ว ๆ นี้

CI/CD & GitHub Actions

ให้หุ่นยนต์ทำงานซ้ำ ๆ แทนเรา: push โค้ดปุ๊บ ระบบทดสอบอัตโนมัติ ผ่านแล้วเว็บอัปเดตเอง — วิธีทำงานมาตรฐานของบริษัทซอฟต์แวร์ยุคนี้

ยังไม่เปิด
10 เร็ว ๆ นี้

Cloud Computing เบื้องต้น

รู้จัก AWS / Google Cloud / Azure: เว็บใหญ่ ๆ ของโลกรันอยู่ที่ไหน, serverless คืออะไร, คิดเงินยังไง และใช้ free tier ให้เป็น

ยังไม่เปิด
11 เร็ว ๆ นี้

Cybersecurity ยุค AI

ช่องโหว่ยอดฮิตที่เว็บจริงโดนบ่อย, API key รั่วบน GitHub, prompt injection และวิธีตรวจความปลอดภัยของโค้ดที่ AI เขียนให้

ยังไม่เปิด
12 เร็ว ๆ นี้

IoT & Edge AI

ต่อโลกจริงเข้ากับโค้ด: ไมโครคอนโทรลเลอร์ + เซนเซอร์ ส่งข้อมูลขึ้นเว็บ สั่งงานผ่านมือถือ และรัน AI บนอุปกรณ์เล็ก ๆ

ยังไม่เปิด
13 เร็ว ๆ นี้

Data + AI วิเคราะห์ข้อมูล

ให้ AI ช่วยอ่านข้อมูลดิบเป็นคำตอบ: Python notebook, ทำความสะอาดข้อมูล, สร้างกราฟ และเล่าเรื่องจากตัวเลขอย่างที่บริษัทต้องการ

ยังไม่เปิด
14 เร็ว ๆ นี้

ออกแบบ UX/UI ยุค AI

ทำไมบางเว็บใช้แล้ว "ลื่น" บางเว็บใช้แล้วหงุดหงิด: หลักการออกแบบพื้นฐาน, Figma, และเวิร์กโฟลว์ design-to-code ด้วย AI

ยังไม่เปิด
15 เร็ว ๆ นี้

โปรเจกต์จริง — สร้างผลงานของตัวเอง

รวมทุกอย่างที่เรียนมา สร้างเว็บแอปจริง 1 ชิ้นตั้งแต่ไอเดียถึง deploy: วางสโคป, ไล่ Milestone, ใช้ AI เป็นคู่หู และเก็บลง portfolio

ยังไม่เปิด
16 เร็ว ๆ นี้

เตรียมสู่โลกทำงาน

เปลี่ยนสิ่งที่เรียนเป็นโอกาสงาน: แต่ง GitHub profile ให้ recruiter หยุดดู, portfolio + resume ยุค AI และการนำเสนอผลงานของตัวเอง

ยังไม่เปิด
